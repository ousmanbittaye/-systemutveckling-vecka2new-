# Fråga 2

GIt-commit är som ett spar knapp i ett projekt men ett smartare version för den spara inte bara filerna utan den sparar också vad som ändrades, när det ändrades och vem som gjorde ändringen. Git-commit är viktigt för att du kan se hur projekten har utvecklats steg för steg och det underlättar för personer som jobbar tillsammans med ett projekt att hålla koll på alla ändringar som har hänt under projektens gång. Man kan också enkel gå tillbaka till ett tidigare commit om något skulle gå fel.
